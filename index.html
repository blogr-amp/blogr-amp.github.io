ampaction(js)

An All-In-One AMP HTML Widget for Blogger theme integrations with AMP HTML

Example: Display Blog Feed

amp-iframe: Displays Blog Feed (default)


 

 <!-- required values -->
?ampactions=feed
&amp;canonicalurl={{BLOG_CANONICAL_URL}}
&amp;canonicalhomepageurl={{BLOG_CANONICAL_HOMEPAGE_URL}}


<!-- optional values -->
&amp;feedlimit=999
&amp;feedresults=5
&amp;feedsummary=160
&amp;feedimage=320

amp-iframe: Displays Blog Related Feed (by Label)


 

<!-- required values -->
?ampactions=feed
&amp;feedrelated=true
&amp;feedrelatedlabel={{BLOG_POST LABEL}}
&amp;canonicalurl={{BLOG_CANONICAL_URL}}
 &amp;canonicalhomepageurl={{BLOG_CANONICAL_HOMEPAGE_URL}}


<!-- optional values -->
&amp;feedlimit=999
&amp;feedresults=5
&amp;feedsummary=160
&amp;feedimage=320

Example: DISQUS Comments

 

<!-- required values -->
?ampactions=disqus
&amp;shortname={{DISQUS_SHORTNAME}}
&amp;url={{BLOG_URL}}
&amp;homepageurl={{BLOG_HOMEPAGE_URL}}
&amp;canonicalurl={{BLOG_CANONICAL_URL}}
&amp;canonicalhomepageurl={{BLOG_CANONICAL_HOMEPAGE_URL}}
