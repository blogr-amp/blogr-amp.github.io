ampaction(js)

An All-In-One AMP HTML Widget for Blogger theme integrations with AMP HTML

Example: Display Blog Feed

amp-iframe: Blog Feed Random Order (default)

Nos Post Display: 3


 
<!-- required values -->
?ampactions=feed
&amp;feedlimit=999
&amp;canonicalurl={{BLOG_CANONICAL_URL}}
&amp;canonicalhomepageurl={{BLOG_CANONICAL_HOMEPAGE_URL}}

<!-- optional values -->
&amp;feedresults=5
&amp;feedsummary=160
&amp;feedimage=320

amp-iframe: Blog Related Feed Random Order (by Label)

Label: Getting Started


 
<!-- required values -->
?ampactions=feed
&amp;feedrelated=true
&amp;feedrelatedlabel={{BLOG_POST LABEL}}
&amp;feedlimit=999
&amp;canonicalurl={{BLOG_CANONICAL_URL}}
&amp;canonicalhomepageurl={{BLOG_CANONICAL_HOMEPAGE_URL}}

<!-- optional values -->
&amp;feedresults=5
&amp;feedsummary=160
&amp;feedimage=320
